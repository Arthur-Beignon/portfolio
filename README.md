# 🌐 Portfolio Website

Welcome to my personal portfolio repository!
This project is a **static website** built with **HTML, CSS, and JavaScript** to showcase my work, projects, and skills.

---

## 🚀 About the Project

This portfolio reflects my work as a developer through a clean, modern, and responsive interface.
It includes both **French 🇫🇷 and English 🇬🇧 versions** of the site.

---

## ✨ Features

* 🌍 Bilingual website (FR / EN)
* 📱 Fully responsive design
* ⚡ Lightweight and fast (no frameworks)
* 🎨 Clean and modern UI
* 🧩 Interactive elements with JavaScript

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript** – Interactivity

## ⚙️ Usage

1. Clone the repository:

```bash
git clone https://github.com/Arthur-Beignon/portfolio.git
```

2. Open the project folder:

```bash
cd portfolio
```

3. Launch the site:

* Open `index.html` (FR) or `index_en.html` (EN) in your browser
